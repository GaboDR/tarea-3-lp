--------------------------------------------------------------------
CONSIDERACIONES AL MOMENTO DE EJECUTAR
--------------------------------------------------------------------
- Todos loa archivos deben de estar en la misma ubicacion para su e-
  jecucion.
- Las interacciones seran por consola, se espera que la entrada sea 
  una de las que se indica; como lo dice el enunciado.
- Cuando se consumen cosas el inventario se consideran como unidades
  (int) y a pesar que se usen fracciones del material, se cosidera 
  el descuento de la unidad entera.
- Quedarse sin combustible no activa el sistema de emergencia, que a
  libertad del jugador ver como resolver esa situacion.
- Los nombres de los planetas son el tipo asociado con una serie de 
  numeros y letras (direccion de memoria) para hacerlo tipo Yavin 4.
- El Centro Galactico se genera correctamente, pero al tener 1% de 
  probablidad de aparecer, casi nunca aparece al generar, recomien-
  do tener una eficiencia alta y hacer un salto grande para que se 
  genere.
